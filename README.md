# AI Meeting Minutes Generator

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

An intuitive, modern web application designed to instantly generate professional, clean, and searchable meeting minutes in Markdown format. Built with React (Next.js), Tailwind CSS, and pure TypeScript for lightning-fast, client-side generation.

This tool helps streamline project management and corporate documentation by providing a user-friendly interface to input meeting details and receive a perfectly structured template in seconds.

## ‚ú® Features

-   **Instant Markdown Generation:** All processing is done client-side for immediate, real-time results. No server, no waiting.
-   **Structured & Intuitive Form:** Cleanly designed fields for all essential meeting details: title, date, attendees, agenda, discussion, and action items.
-   **Dynamic Action Items:** Easily add or remove action items, including task description, owner, and due date.
-   **Live Preview Panel:** See the formatted Markdown output update instantly as you type.
-   **One-Click Copy & Download:** Quickly copy the raw Markdown to your clipboard or simulate a DOCX file download.
-   **Fully Responsive Design:** A seamless and comfortable experience across desktop, tablet, and mobile devices.
-   **Immersive UI/UX:** Features a beautiful, animated galaxy background and a modern, professional interface.
-   **SEO Optimized:** Includes a comprehensive 3,500-word SEO article and full JSON-LD schema to maximize search engine visibility.
-   **Privacy-First:** Your data is your own. No information is ever sent to a server. All meeting details stay securely in your browser.
-   **Built-in Modals:** Quick access to helpful information like a "How to Use" guide, privacy policy, and contact details.

## üöÄ Getting Started

To run this project locally, follow these simple steps.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine (v16 or higher is recommended).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/ai-meeting-minutes-generator.git
    cd ai-meeting-minutes-generator
    ```
2.  **Install dependencies:**
    (Note: This project uses CDN imports for React, so no `npm install` is needed for core libraries. If you add other dependencies, you would run `npm install` here.)

3.  **Run the application:**
    You can serve the `index.html` file using a simple local server. A common tool for this is `live-server`.
    ```bash
    # If you don't have live-server, install it globally:
    npm install -g live-server

    # Run the server from the project root
    live-server
    ```
    The application will open automatically in your default browser.
